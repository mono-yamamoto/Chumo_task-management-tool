# カラーシステム設計リファレンス

## トークンの2層構造

### プリミティブカラー（Primitive Colors）

- 基礎となる生のカラーコード（例: `teal-600: #008B8A`）
- 色相ごとに明度の段階（50〜950）で展開
- この層は「色の在庫」であり、直接UIに使わない

### セマンティックカラー（Semantic Colors）

- UIの意味・用途に基づく名前（例: `text-primary`, `bg-surface`, `border-default`）
- プリミティブカラーを参照する
- テーマ切り替えは**セマンティック層の参照先を変えるだけ**で実現

```
例:
text-primary:
  light → neutral-900 (#111827)
  dark  → neutral-50  (#F9FAFB)
```

## コントラスト比（WCAG 2.2）

### 必須基準

| 対象                                   | 最低コントラスト比 | 備考                     |
| -------------------------------------- | ------------------ | ------------------------ |
| 通常テキスト（<18px / <14px bold）     | **4.5:1**          | 本文、ラベル等           |
| 大きいテキスト（>=18px / >=14px bold） | **3:1**            | 見出し等                 |
| UI要素（ボーダー、アイコン等）         | **3:1**            | ボタン境界、フォーム枠等 |
| 装飾・無効状態                         | 基準なし           | disabledは対象外         |

### Chumoブランドカラーの注意点

- `#008B8A`（ブランドカラー）は白背景で**約4.1:1** → 通常テキストの4.5:1を満たさない
- `#007170`（teal-700）で**約4.7:1** → 通常テキストOK
- ブランドカラーはボタン背景・大きいUI要素に使い、テキストには700以上を使う

### 実践的な使い分け

```
ブランドカラー #008B8A の使い方:
  OK: ボタン背景（白テキストを上に載せる）、大きい見出し、アイコン
  NG: 白背景上の通常サイズテキスト（4.5:1未達）

テキスト用: #007170 (teal-700) 以上を使う
```

## カラーモデルの選択

| モデル    | 特徴                                   | 推奨度             |
| --------- | -------------------------------------- | ------------------ |
| HSB       | 直感的。デザイナー向き                 | デザインツール内OK |
| HSL       | 明度概念あるが知覚と不一致             | 避ける             |
| LCH       | 知覚的明るさと一致。青で色相シフト発生 | まあまあ           |
| **OKLCH** | LCH改良版。CSS対応済み                 | **推奨**           |
| HCT       | Material Design 3採用。WCAG基準重視    | 良い               |

- パレット生成時はOKLCHベースが理想的（知覚的に均等な明度ステップが作れる）
- CSS実装では `oklch()` が使える

## デジタル庁デザインシステムの構造（参考）

### キーカラー層（ブランド表現用）

- **Primary**: CTA・主要アクション。背景との4.5:1必須
- **Secondary**: Primaryと同色相・異明度の副次的要素
- **Tertiary**: Secondaryと反対の明度
- **Background**: 非白黒の背景色

### 機能色層（UI状態表現用）

- **共通カラー**: グレースケール（テキスト・枠線・背景）
- **機能カラー**: リンク（青=未訪問/紫=訪問済）
- **セマンティック**: Success(緑) / Error(赤) / Warning(黄〜橙)
- **アクセント**: 注意引き用（多用回避）

### 設計原則

- セマンティックカラーは**色相を保てば、明度・彩度は調整OK**
- テキスト用は4.5:1以上、非テキスト要素は3:1以上

## プリミティブカラーのスケール設計

### 基本構成

- 1色相につき11段階: 50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950
- 50: ほぼ白（背景用tint）
- 500: 中間明度
- 950: ほぼ黒

### ブランドカラーの配置

- ブランドカラーの明度に応じてスケール内の位置を決める
- `#008B8A` は明度が低い（HSL L≈27%）ので600に配置
- 500以下を明るい派生色、700以上を暗い派生色として展開
